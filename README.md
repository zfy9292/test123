# 赵奉禹个人网站 / Fengyu Zhao Personal Website

这是一个可直接上线的静态个人网站升级版，风格定位为：学术、国际化、数字商务、跨境视野、实践导向。

## 文件结构

- `index.html`：网站主页面
- `styles.css`：视觉样式、响应式布局、悬浮效果、动画效果
- `script.js`：中英双语切换、滚动进度条、入场动画、导航高亮、卡片轻微 3D 悬浮、邮箱复制
- `assets/profile.jpeg`：个人照片

## 新增功能

- 中英双语切换
- 滚动进度条
- 页面内容滚动入场动画
- 导航栏当前栏目高亮
- 研究卡片、论文卡片、项目卡片、链接卡片悬浮提升效果
- 个人卡片与联系卡片轻微 3D 视差效果
- 鼠标光晕背景效果
- 返回顶部悬浮按钮
- 邮箱一键复制
- 外部链接区：学校官网、学院官网、代表论文页面、邮件入口

## 本地预览

直接双击 `index.html` 即可在浏览器打开。

## 部署建议

可部署到：

1. GitHub Pages
2. 学校个人主页服务器
3. 个人域名服务器
4. 任意静态网站托管平台
